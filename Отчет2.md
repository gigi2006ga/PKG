Отчёт по лабораторной работе №2

Тема: Чтение информации из графических файлов

Цель работы:
Изучение методов получения информации о графических файлах (формат, размер, глубина цвета, сжатие) с использованием веб-технологий.

Ход работы:

Разработан интерфейс веб-приложения, напоминающий вид папки:

Левая панель отображает текущую папку;

Правая панель отображает загруженные изображения в виде карточек с информацией.

Реализована возможность загрузки отдельных файлов и папок с изображениями.

Для каждого изображения получаются следующие параметры:

Имя файла;

Расширение (формат);

Размер изображения (ширина × высота);

Глубина цвета (битность);

Тип сжатия (для JPEG примерный процент, для PNG/GIF/BMP указано сжатие или отсутствие).

Реализован прогресс-бар, показывающий процесс обработки файлов.

Использован JavaScript и библиотека ExifReader для анализа метаданных изображений.

Используемые технологии:

HTML, CSS для создания интерфейса;

JavaScript для обработки файлов, вычисления информации и динамического обновления интерфейса;

ExifReader.js для чтения метаданных изображений;

Браузер для запуска приложения (локально).

Результаты:

Веб-приложение успешно загружает и анализирует изображения;

Информация по каждому изображению выводится на карточках;

Прогресс обработки файлов отображается динамически;

Визуально интерфейс имитирует вид папки, что делает работу удобной и наглядной.

Выводы:

Получен опыт работы с API файлов в браузере;

Изучено, как получать метаданные изображений и вычислять параметры, такие как глубина цвета и сжатие;

Освоены методы динамического создания и обновления DOM элементов для отображения данных;

Разработанное веб-приложение демонстрирует эффективный способ визуализации информации о файлах для пользователя.
