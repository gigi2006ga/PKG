Отчёт по лабораторной работе
Конвертация цветовых моделей (RGB ↔ CMYK ↔ HSV)
1. Цель работы

Разработать графическое приложение на Python с использованием tkinter для конвертации цветов между моделями RGB, CMYK и HLS. Реализовать визуальное отображение выбранного цвета и возможность ввода значения цвета вручную (через слайдеры, поля ввода и HEX-код).

2. Задачи

Изучить алгоритмы преобразования цветов между моделями RGB, CMYK и HLS.

Реализовать функции преобразования:

RGB → CMYK и CMYK → RGB

RGB → HLS и HLS → RGB

RGB → HEX и HEX → RGB

Создать графический интерфейс с тремя блоками для каждой цветовой модели.

Добавить визуальный превью и возможность выбора цвета через стандартный диалог colorchooser.

Обеспечить корректное взаимодействие между различными моделями при изменении значений пользователем.

3. Используемые инструменты

Python 3.x

Библиотека tkinter для графического интерфейса

Стандартные библиотеки: math, messagebox, colorchooser

4. Основная часть (код программы)

Здесь приводится сам код программы или его ключевые фрагменты с пояснениями

Пример ключевой функции: RGB → CMYK
def rgb_to_cmyk(r, g, b):
    if (r, g, b) == (0, 0, 0):
        return 0.0, 0.0, 0.0, 100.0
    r_p, g_p, b_p = r / 255, g / 255, b / 255
    k = 1 - max(r_p, g_p, b_p)
    c = (1 - r_p - k) / (1 - k)
    m = (1 - g_p - k) / (1 - k)
    y = (1 - b_p - k) / (1 - k)
    return round(c * 100, 2), round(m * 100, 2), round(y * 100, 2), round(k * 100, 2)

5. Описание интерфейса

Блок RGB: слайдеры и поля для R, G, B (0–255)

Блок CMYK: слайдеры и поля для C, M, Y, K (0–100%)

Блок HLS: слайдеры и поля для H (0–360), L, S (0–100)

Превью: прямоугольник с выбранным цветом

Дополнительно: ввод HEX-кода, копирование в буфер обмена, выбор цвета через диалог

6. Примеры работы

Установка цвета RGB = (255, 0, 0)

CMYK = (0, 100, 100, 0)

HLS = (0, 50, 100)

HEX = #FF0000

Превью отображает ярко-красный цвет

7. Выводы

Приложение позволяет конвертировать цвета между основными моделями корректно.

Пользователь может изменять значения в любой модели, изменения автоматически отображаются во всех остальных моделях.

Реализация визуального превью и HEX-кода повышает удобство использования.

Были изучены алгоритмы преобразования RGB ↔ CMYK и RGB ↔ HLS, включая нюансы вычисления оттенков и насыщенности.
